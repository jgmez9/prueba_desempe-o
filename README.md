This is a simple system with which you, as the user,   
can organize your routine tasks in a way that makes   
them easy and quick to manage.  

Its features include:

Register tasks  
View task list  
Search for a specific task  
Update tasks  
Delete tasks  

THIS IS HOW WE VIEW THE MENU  

===System_Managment===  
           1.New task  
           2.Check task  
           3.Search   
           4.Update task  
           5.Delete task
